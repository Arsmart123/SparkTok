1. use "npm install" to install all the packages
2. use "npm run compile" to compile all the jsx into js file
3. use "npm start" to start a 8000 port
4. Go to browser, open "localhost:8000" and refresh the page. You will see a UI page!

The title is Spark Tok, and the background is dynamic, and will change every time you refresh the page. Also, below shows two tabs, left one is the abstract and the right one is the author info. Upper right locates a "next paper" button. In future version, you will see a new paper when you press this button.
